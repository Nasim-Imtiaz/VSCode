# VSCode Shorcuts For MacOS &#63743;
<code>&#8984; + T</code> Prompt a window for search file. 
<br>
<code>&#8984; + F</code> Find something on that file. 
<br>
<code>&#8984; + &#8679; +F</code> Find something thorugh all files.
